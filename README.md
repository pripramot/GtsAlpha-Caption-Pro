# GtsAlpha Caption Pro

**เครื่องมือ Python ดึงคำบรรยาย YouTube แปลไทย + พากย์เสียงไทยอัตโนมัติ**

เครื่องมือสำหรับงานวิจัยและการตลาดยุคใหม่

### ฟีเจอร์
- ดึงคำบรรยายภาษาอังกฤษจาก YouTube
- แปลเป็นภาษาไทย
- บันทึกไฟล์ .srt (มีเวลา)
- สร้างเสียงพากย์ไทย (.mp3)
- GUI ภาษาไทย 100%

### วิธีใช้งาน
1. `pip install -r requirements.txt`
2. `python src/GtsAlpha_Caption_Pro_Thai_Final.py`
