เทคโนโลยีที่ใช้

เลือกสิ่งที่ดีที่สุดสำหรับ Desktop App — เบา เร็ว และปลอดภัย

🦀 Backend — Rust ประสิทธิภาพสูง ปลอดภัย ไม่มี GC

🍎 WKWebView (macOS) WebView ที่ใช้ใน Safari — Native Performance
🐧 WebKitGTK (Linux) WebKit สำหรับ Linux Desktop
Tauri v2
❌ ไม่ฝัง Chromium — ต่างจาก Electron อย่างไร?
Tauri ใช้ WebView ของระบบปฏิบัติการที่มีอยู่แล้ว จึงไม่ต้องดาวน์โหลดหรือรวม Chromiumไปกับโปรแกรม ทำให้ขนาดไฟล์ติดตั้งของ Tauri อยู่ที่ ~3–10 MB เทียบกับ Electron ที่มักเกิน 120 MB นอกจากนี้ยังใช้ RAM น้อยกว่า เนื่องจากไม่ต้องรัน Chromium instance แยกต่างหาก
การติดตั้ง

เริ่มต้นใช้งาน

ติดตั้ง dependency ไม่กี่ขั้นตอนแล้วรันโปรแกรมได้ทันที

1

ติดตั้ง Rust Toolchain

ดาวน์โหลดจาก rustup.rs หรือรัน:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

2

ติดตั้ง Node.js & npm

ดาวน์โหลดจาก nodejs.org (แนะนำ LTS v20+)

3

Clone repository และติดตั้ง dependencies

git clone https://github.com/pripramot/GtsAlpha-Caption-Pro.git
cd GtsAlpha-Caption-Pro && npm install

4

รันในโหมด Development

npm run dev — เปิดหน้าต่างแอปพร้อม Hot Reload

5

Build เป็นไฟล์ติดตั้ง

npm run build — สร้าง .exe / .dmg / .AppImage ใน src-tauri/target/release/bundle/
